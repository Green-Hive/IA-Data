# First Step

## Install venv first 

```
pip install virtualenv
```


## Create virtual environement

```
python -m venv local
```

## Activate virtual Environement

```
    MacOS
        $ source local/bin/activate

    Windows
        $ .\local\Scripts\activate

    Linux
        $ source local/bin/activate
```

## Install depedencies

```

pip install -r ./DB/API/requirements.txt
```

# Enjoy :)
