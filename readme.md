# First Step

## Install venv first 

```
pip install virtualenv
```


## Create virtual environement

```
python -m venv env
```

## Activate virtual Environement

```
    MacOS
        $ source env/bin/activate

    Windows
        $ .\env\Scripts\activate

    Linux
        $ source env/bin/activate
```

## Install depedencies

```

pip install -r /DB/API/requirements.txt
```

# Enjoy :)
